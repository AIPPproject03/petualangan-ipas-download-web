Cara Download & Install 📲

Ikuti langkah-langkah berikut — mudah, cepat, gratis!

1
🔓 Aktifkan 'Sumber Tidak Dikenal' di HP kamu
Perlu diizinkan untuk install APK

Sebelum install, HP Android perlu diizinkan untuk memasang aplikasi dari luar Play Store.

  1. Buka Pengaturan (Settings) di HP kamu
  2. Cari menu Keamanan atau Privacy
  3. Aktifkan "Instal aplikasi dari sumber tidak dikenal" atau "Unknown sources"
  4. Pilih browser/file manager kamu → Izinkan
⚠️ Ini aman dilakukan. Aplikasi Petualangan IPAS sudah diuji dan terpercaya.
2
⬇️ Download File APK Petualangan IPAS
Download dari link di bawah

Klik tombol di bawah untuk mulai download file APK aplikasi Petualangan IPAS.

⬇️ DOWNLOAD APK SEKARANG
📦 Format: .apk · Android 5.0+ · Siap diunduh

File akan tersimpan di folder Downloads HP kamu.

3
📲 Buka & Install File APK
Proses instalasi hanya butuh beberapa detik
  1. Buka aplikasi File Manager atau Downloads di HP kamu
  2. Cari file PetualanganIPAS.apk
  3. Tap file tersebut → muncul popup instalasi
  4. Tap "Install" dan tunggu beberapa detik
  5. Tap "Buka" setelah selesai — selamat berpetualang! 🎉
✅ Instalasi selesai! Ikon 'Petualangan IPAS' akan muncul di layar HP kamu.
4
🚀 Mulai Petualanganmu!
Daftar dan mulai belajar
  1. Buka aplikasi Petualangan IPAS
  2. Masukkan Nama Lengkap dan NISN kamu
  3. Pilih Kelas (4 atau 5)
  4. Pilih BAB yang ingin dipelajari
  5. Tonton video, lalu kerjakan Misi — Selamat berpetualang! 🌟

Pertanyaan Umum ❓

Apakah aplikasi ini gratis?
+
Ya, 100% gratis untuk semua siswa! Tidak ada biaya tambahan atau langganan tersembunyi. Aplikasi ini dikembangkan khusus untuk mendukung pembelajaran IPAS siswa kelas 4 dan 5 SD.
HP apa yang bisa menggunakan aplikasi ini?
+
Semua HP Android versi 5.0 ke atas. Jadi hampir semua HP Android yang masih aktif bisa menjalankan Petualangan IPAS dengan lancar.
Anak saya tidak punya NISN, bagaimana?
+
Hubungi wali kelas atau guru untuk mendapatkan NISN. NISN adalah nomor unik setiap siswa yang diberikan oleh sekolah. Wali kelas pasti memiliki data ini.
Apakah butuh internet?
+
Internet dibutuhkan untuk login dan menonton video. Quiz bisa dikerjakan offline setelah video sudah didownload. Ini membantu siswa yang memiliki koneksi internet terbatas.
Apakah aman untuk anak-anak?
+
Ya! Aplikasi ini dirancang khusus untuk siswa SD kelas 4 dan 5, tanpa iklan dan konten berbahaya. Semua fitur sudah diuji untuk memastikan keamanan dan kenyamanan anak dalam belajar.
